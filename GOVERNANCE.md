# 熔炉治理章程

## 一、总则

熔炉 (ForgeShell) 是一个社区驱动的开源项目。本治理章程规定了项目决策权、贡献者层级、悬赏机制与赞助管理规则。

## 二、锻师会层级

### 大锻师
- 对项目拥有最终决策权和否决权
- 审核锻师会合并的 PR
- 管理悬赏平台和资金

### 锻师
晋升条件（满足其一即可）：
- 合并 10 个以上 PR
- 在社区分享的 SOP 帮助超过 500 人次

权限：
- PR 投票权（≥2 位锻师批准方可合并）
- 路线图提议权
- 悬赏认领权

### 学徒
晋升条件：
- 至少 1 个 PR 被合并 或 1 条 SOP 被采纳

权限：
- 参与社区讨论
- 提交 PR 和 SOP

## 三、悬赏机制

1. 任何人可在悬赏榜发布任务并预付赏金
2. 锻师认领任务并完成后，由锻师会审核
3. 审核通过后赏金自动发放给完成者
4. 平台服务费：¥500 以下免费，¥500 以上统一 5%
5. 服务费归大锻师管理，用于项目发展

## 四、赞助规则

1. 国内用户通过爱发电赞助，国际用户通过 GitHub Sponsors
2. 赞助金由发起人管理，用于项目发展及运维
3. 不公开赞助者名单、金额及支出明细
4. 不定期在社区发起悬赏活动推动关键功能

## 五、荣誉体系

- 徽章等级：初识进化 → 锻火之心 → 千锤百炼 → 不朽锻匠
- 锻师认证证书（可打印验证）
- 熔炉史册永久载入贡献者

## 六、隐私承诺

1. 复盘上传仅含脱敏策略，不上传代码、路径、变量名、API Key
2. 分享弹窗明确列出不上传的内容，支持预览脱敏数据

## 七、身份认证与打款

1. 社区身份通过 Gitee / GitHub OAuth 绑定，不建独立账户体系
2. 锻师在社区面板绑定支付宝/微信/PayPal 用于收赏金
3. 每季度系统自动生成付款清单，大锻师审核打款
