# 熔炉 (ForgeShell) 项目状态报告

> 版本：v0.12.1 | 日期：2026-05-20 | 开发周期：3 天

## 一、项目概述

**熔炉** 是一款面向中文开发者的开源 AI 编程终端，对标 Claude Code 和 Reasonix，在成本、隐私、社区进化三个维度形成差异化优势。

- 代码仓库：https://gitee.com/forgemaster/forge-shell
- 社区网站：https://forgeshell.cn
- 许可证：AGPLv3（代码） + CC BY-SA 4.0（数据）
- 技术栈：Rust 1.85+ · Axum · Ratatui · DeepSeek V4 API

## 二、核心架构

```
用户电脑
├── Web UI (localhost:9527) — 紫金配色，15 工具快捷面板
├── TUI — 全中文终端界面，三模式切换
├── MCP (/api/mcp) — 兼容 Anthropic 生态
│
引擎层：
├── 四层缓存：L1恒定提示词 → L2项目指纹 → L3会话压缩 → L4当前指令
├── Prefix Cache 优化：系统提示词一字不动，DeepSeek 命中率 90%+
├── 模型路由：规划→Pro / 极速→Flash / 助手→自动
├── 子任务路由：复杂任务拆解，读 Flash 写 Pro，成本降 60%
├── 进化引擎：匿名采集→反思提炼SOP→天工阁匹配
├── 智能沙箱：按文件类型(.rs/.toml/.md)动态放行命令
│
数据全本地存储（%LOCALAPPDATA%/forge-shell/），零上传
```

## 五、竞争优势（结构性壁垒）

1. **成本（1/21 of CC）**：四层本地缓存 + Prefix Cache + 子任务级模型路由
2. **隐私**：全部数据本地存储，不上传任何代码、路径、Key
3. **社区进化**：天工阁 SOP 积累 + 悬赏榜激励 + CC BY-SA 数据许可
4. **AGPLv3 防抄袭**：大厂修改必须开源，SOP 数字指纹溯源
5. **MCP 生态兼容**：零成本接入 Anthropic MCP 工具生态
6. **中文深度优化**：国内镜像 + 中文错误解析 + 国产模型自动切换
