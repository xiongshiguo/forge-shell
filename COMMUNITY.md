# 熔炉社区机制

## 进化闭环

```
用户使用 → 脱敏复盘 → 一键分享 (Ctrl+S) → 公共池
                                              ↓
用户变聪明 ← 自动匹配 SOP ← 天工阁 ← 反思引擎提炼
```

## 经验熔池

- 用户提交的脱敏复盘汇聚于此
- 反思引擎每周提炼优秀策略
- 自动匹配到相关场景

## 天工阁（SOP 库）

- 由反思引擎从经验熔池提炼
- 社区锻师可手动创建和维护
- 使用时自动匹配当前任务

## 悬赏榜

- 任何人都可发布悬赏任务
- 预付赏金托管在平台
- 锻师会审核完成后发放

## 锻师会

- 项目核心贡献者治理体系
- 负责 PR 审核、SOP 维护、悬赏审核
- 详见 GOVERNANCE.md

## 加入社区

1. Fork 项目仓库
2. 提交 PR 或分享 SOP
3. 成为学徒 → 锻师
